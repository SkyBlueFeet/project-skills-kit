# HTML 代码规范（工程执行版）

## 1. 语义与结构

- 优先语义化标签（`header/main/nav/section/article/footer`）。
- DOM 层级保持扁平，避免无意义嵌套。
- 结构、样式、行为分离，不在 HTML 中堆积内联脚本与样式。

## 2. 可访问性（强制）

- 表单控件必须有可关联的 `label`。
- 图片必须有有效 `alt` 文本（纯装饰除外）。
- 可交互元素必须可键盘访问。

## 3. 维护性

- 组件区块命名稳定，避免动态拼接不可追踪的 id/class。
- 模板变量命名可读，避免单字母。

## 4. 检查建议

- 建议使用 HTML 校验工具与可访问性检查。
